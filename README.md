# E-Commerce Project

A simple e-commerce website with:

- Product listing
- Cart system
- Checkout page
- Stripe payment integration
- SQLite order storage (Node.js backend)

## Technologies

- HTML, CSS, JavaScript
- Node.js + Express
- Stripe API
- SQLite (better-sqlite3)

## How to run

1. Start backend:
   cd server
   node server.js

2. Start frontend:
   Use Live Server or any static server.
